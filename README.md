# 🐾 Paw Map (포우맵)

RP 중 장소를 기록하고, 장소별 실시간 커뮤니티 피드를 생성하는 SillyTavern 확장입니다.

> v0.9.2부터 모든 자동 장소 감지가 `autoDetect` 마스터 스위치(기본 OFF) 뒤로 비활성화되어, 기본값은 완전 수동 입력입니다. 장소는 직접 추가/수정/삭제하고, 드래그→이벤트 기록은 설정에서 on/off 할 수 있습니다.

---

## 기능

### 장소 관리
- 장소 직접 추가/수정/삭제 (한국어/영어)
- Leaflet 기반 실제 세계지도
- 장소별 노드맵 약도
- 걸어서 거리 계산

### 실시간 커뮤니티 피드
장소마다 트위터 스타일 짧은 포스트 피드를 생성합니다.

- 포스트 4~12개 생성 (설정으로 조절)
- 장소 해시태그 기반 주민 반응
- 현지 음식·문화·언어 자동 반영
- 목격담 트윗 (최근 RP 맥락 활용)
- 답글, 이미지 자동 첨부
- 여러 톤 섞어서 생성 (담백/약한 감성/폭발 감성)

### 현지 정보 보강 (선택)
- **OFF**: LLM 학습 데이터만 사용
- **Nominatim**: 장소 주변 500m POI 정보를 프롬프트에 추가 (무료)
- **Google Search Grounding**: 실시간 웹 검색 결과 반영 (유료)

### NPC 관리
- 장소별 등장 NPC 자동 감지
- 호감도, 성격, 관계 메모 관리

### 기타
- 채팅 드래그 → 요약 후 이벤트 저장
- 모바일 대응 디버그 로그 뷰어
- 그룹챗 다인 캐릭터 대응

---

## 설정

### API 키 등록
설정 → LLM API 키에서 제공자 선택 후 키 입력:

- Google AI Studio: https://aistudio.google.com/apikey
- Vertex AI (Express mode 또는 서비스 계정 JSON)
- OpenAI / OpenRouter / Claude

### 추천 모델
- **Gemini 2.5 Flash** (기본 추천, 빠르고 저렴)
- Gemini 2.5 Pro (느리고 토큰 많이 사용, 필요시에만)
- Gemini 2.5 Flash Lite (최저 비용)

### 옵션
- **생성 분량**: 가벼움(4~5개) / 기본(7~9개) / 풍성함(10~12개)
- **현지 정보 보강**: OFF / Nominatim / Google Grounding

## 비용 참고

Gemini 2.5 Flash 기준 대략:

| 분량 | 1회 | 월 450회 |
|---|---|---|
| 가벼움 | ~6원 | ~2,700원 |
| 기본 | ~14원 | ~6,300원 |
| 풍성함 | ~22원 | ~9,900원 |

Google AI Studio 무료 티어는 일 1,500회까지 지원되어 대부분 유저는 무료로 사용 가능합니다.

Google Search Grounding은 1,000회당 $35 (별도 유료).

---

## 문제 해결

에러 발생 시:
1. 설정 → 🐛 마지막 LLM 응답 보기
2. 📋 복사 버튼으로 로그 복사
3. 로그와 함께 리포트

모바일에서도 F12 없이 디버그 가능합니다.

- 랜덤 생성 다양성 강화

### v0.9.6
- 🎲 터줏대감 랜덤 생성 — 장소 수정창에서 버튼 하나로 그 장소에 어울리는 단골/동물을 LLM이 랜덤 생성 (사람/동물 무작위)
  - 생성된 터줏대감은 프로필(호감도·소개·성격·관계)에서 수정 가능, 목록 항목 탭하면 프로필 열림
- 터줏대감 등장 강화 — "이 장소의 터줏대감(단골/거주 동물)이니, 장면이 여기서 벌어지면 실제로 등장·인사·반응하게" 로 주입 강화 (동물은 물리적으로 나타나도록)

### v0.9.5
- 장소 수정창을 풀스크린 오버레이로 전환 (덮어쓰기 + 내부 스크롤 + 헤더에 닫기/💾 저장)
- 메모 칸 역할 분리
  - 💭 내 메모 = 항상 비공개 (AI 반영 안 됨)
  - 🎬 반영 노트 = RP 반영 모드 따라 캐릭터/연출에 전달
- 커뮤니티 = "이 장소에 대한 소문/분위기"로 주입 (장면 배경에 녹임)
- NPC = 실재 엔티티로 등장 주입 (사람/동물 분리, 동물은 장면에 물리적으로 등장)
- 🔒 끔 장소는 커뮤니티·NPC도 주입 제외 (캐릭터가 모르는 장소)

### v0.9.4

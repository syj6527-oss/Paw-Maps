# 🐶 World Tracker v0.4.0

**RP하면 알아서 채워지는 자동 로어북**

채팅만 하면 장소가 자동으로 지도에 정리되고, AI가 그 정보를 기억해서 대화에 활용합니다.
설정할 것도 외울 것도 없어요. 그냥 RP 하세요!

---

## 이런 분들에게 추천해요

- RP 하다가 "여기 전에 왔던 곳인데..." 하고 까먹는 분
- 캐릭터가 어디있는지, 어디서 뭘 했는지 정리하고 싶은 분
- 구글맵 느낌으로 RP 세계관을 꾸미고 싶은 분
- AI가 장소를 기억하고 분위기에 맞게 대화했으면 하는 분

---

## ✨ 주요 기능

### 🔍 장소 자동 감지
채팅만 하면 장소가 자동 등록됩니다. 한국어, 영어 모두 지원.
> "카페로 향했다" → 📍 카페 등록!
> "headed to the market" → 📍 Market 등록!

### 🗺️ 약도 + 실제 지도 (Paw Maps)
- **약도**: 장소 관계를 한눈에 보는 노드 맵
- **Paw Maps**: 실제 지도 위에 핀으로 표시 (구글맵 스타일)
- 주소 검색 → 실제 좌표에 배치
- 핀 터치 → 장소 상세 정보 (바텀시트)

### 📍 자동 주소 + 거리 계산 (v0.4.0 NEW!)
첫 장소의 주소만 검색하면, 이후 장소들은 알아서 근처에 배치됩니다.
- 새 장소 → 기존 장소 근처에 자동 좌표 배치
- 좌표 기반 실제 거리 자동 계산 (도보 몇 분)
- 주소도 자동 부여!

예) "SAS 막사"에 Hereford 주소를 검색하면,
이후 AI가 "카페에 갔다"고 하면 → 카페가 Hereford 시내에 자동 배치!

### 🏠 내부 장소 (장소 계층) (v0.4.0 NEW!)
집 안의 거실, 부엌, 방 같은 세부 장소를 자동으로 구분합니다.
- AI가 "거실로 갔다" → 집 > 거실 (지도에 핀 추가 안 됨!)
- 각 방마다 독립적인 이벤트 기록
- 바텀시트 "내부" 탭에서 방 목록 확인
- 수정창에서 방 추가/삭제/메모 가능

### 🤖 AI 프롬프트 자동 주입 (v0.4.0 강화!)
현재 위치, 주소, 방문 기록, 이벤트를 AI에게 자동으로 알려줍니다.
AI가 이 정보를 읽고 자연스럽게 대화에 반영해요.

포함 정보:
- 현재 장소 이름 + 주소
- 장소 타입 (카페, 마트, 집 등 자동 감지)
- 방문 횟수 + 첫 방문 날짜
- 최근 이벤트 (날짜 포함)
- 주변 장소 + 거리 + 주소
- 내부 장소 (거실, 부엌 등)
- 세계관 지역 자동 추론

### 🔀 AI 중복 등록 방지 (v0.4.0 NEW!)
유저가 "마트"라고 하고 AI가 "창신동 슈퍼마켓"이라고 하면,
같은 곳인지 자동으로 물어봅니다.
- 같은 곳이면 → 별칭으로 추가
- 다른 곳이면 → 새로 등록

### 📊 바텀시트 (구글맵 스타일)
장소를 탭하면 상세 정보가 나옵니다.
- **개요**: 분위기 카드, 방문 통계, 주변 장소, 리뷰 미리보기
- **이벤트**: 자동 감지된 사건들 (키스, 싸움, 약속 등)
- **리뷰**: AI가 만든 캐릭터 시점 리뷰 (구글맵 스타일)
- **내부**: 건물 안 세부 장소 목록

### ⭐ AI 캐릭터 리뷰
장소마다 AI가 RP 캐릭터 시점의 리뷰를 만들어줍니다.
- 방문 횟수에 따라 1~10개
- 캐릭터 이름, 성격, 별점 랜덤
- RP 이벤트가 리뷰에 반영!

### 📝 이벤트 자동 기록
RP에서 일어나는 사건을 자동으로 감지하고 기록합니다.
- 80+개 키워드 자동 감지 (한/영)
- AI가 분위기에 맞는 요약 생성
- 텍스트 드래그 → 📝 버튼으로 수동 저장도 가능

### 🕐 타임라인
날짜별 이동 기록을 한눈에 볼 수 있습니다.
- 오늘은 펼침, 과거는 접힘 (아코디언)
- 이동 경로 + 이벤트 표시

### 🔖 내 페이지 + 태그 (v0.4.0 NEW!)
장소를 카테고리별로 정리할 수 있습니다.
- 💜 즐겨찾기 / ⭐ 별표 / 🚩 가고싶은곳 / 🧳 여행계획
- 바텀시트 🔖 버튼으로 태그 토글
- 미방문 장소도 미리 등록 가능

### 🌍 세계관 이어가기 (v0.4.0 NEW!)
새 채팅을 만들어도 같은 캐릭터의 세계관을 유지합니다.
- 설정에서 토글 ON → 캐릭터 기반 저장
- 새 채팅에서도 장소, 거리, 이벤트 그대로!

### 🏰 판타지 모드
판타지 세계관용 아이콘과 분위기로 전환됩니다.

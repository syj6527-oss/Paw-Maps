# 🐶 World Tracker (World Map) v0.2.1-beta

**SillyTavern RP 월드 트래커 확장 프로그램**

AI 응답에서 장소를 자동 감지하고, 노드 약도 + 실제 지도에 표시하는 RP 세계관 관리 도구입니다.

---

## ✨ 주요 기능

### 🔍 자동 장소 감지
- AI 응답과 유저 입력에서 **장소 자동 인식** (한국어/영어)
- 이동 동사 기반 감지: "카페로 향했다", "headed to the market"
- USER/AI 감도 분리 — AI는 엄격 모드로 오탐 방지
- 기존 채팅에 설치 시 → **최근 메시지 스캔 + 승인 UI**

### 🗺️ 노드 약도
- 거리 슬라이더(1~10) 기반 **자동 배치**
- 색상 코딩: 🟣현재 / 🔵자주 / 🟠일반 / 🟡새곳
- 경로선 위 거리 점(●●●○○) + 텍스트 표시
- 스프링 시뮬레이션으로 자연스러운 배치
- 롱프레스로 노드 이동 가능

### 🌍 실제 지도 (Leaflet)
- OpenStreetMap 기반 실제 지도
- **주소 검색** → Nominatim 지오코딩 → 핀 배치
- **앵커 원형 분포** — 한 장소 주소 검색 시 나머지 장소 자동 배치 (30~150m)
- 균일 크기 커스텀 마커 (32x32px, 색상 코딩)
- 롱프레스로 현재 위치 이동

### 🤖 AI 프롬프트 주입
- 현재 위치, 상태, 방문 횟수 자동 주입
- **근처 장소 + RP 거리 정보** 포함
- AI가 이동 현실성을 유지하도록 가이드

```
[🐶 World Tracker]
📍 Scene: 집
📊 Visit #5
💭 편안한 우리 집
🚶 Last: 편의점 → 집
📌 Nearby:
- 편의점 (도보 2분)
- 체육관 (도보 10분)
- 공원 (차량 필요)
[/World Tracker]
```

### 📏 거리 시스템
- **자유 텍스트** + **레벨 슬라이더 (1~10)** 병행
- RP 친화적 라벨: 바로 옆 / 매우 가까움 / 도보권 / 차량 필요 / 다른 지역
- "골목 두 개 건너", "지하 터널 5분" 같은 세계관 표현 가능

### 🐾 추가 기능
- 커스텀 알림 시스템 (토스트)
- 한/영 이중 등록 방지 (집↔Home 사전 28그룹)
- 별칭 관리 (쉼표 구분)
- 메모 + 상태 기록
- 이동 히스토리 (삭제 가능)
- 기억 모드: 💎완벽(영구) / 🌿자연(시간 경과 흐림)

## 🎮 사용법

### 처음 사용
1. RP 채팅 시작 → 자동으로 장소 감지
2. 감지된 장소 **승인 UI**에서 확인/수정/등록
3. 패널에서 장소 터치 → 메모/상태/별칭/거리 설정

### 약도 만들기
1. 장소 2개 이상 등록
2. 장소 터치 → 거리 슬라이더로 다른 장소와의 거리 설정
3. **🗺️ 약도** 버튼 클릭 → 자동 배치!

### 실제 지도 핀 찍기
1. 장소 터치 → 팝오버 하단 **🔍 주소 검색**
2. "서울 창신동" 또는 "Brooklyn" 입력 → 검색
3. 결과 터치 → **자동으로 Leaflet 전환 + 핀 배치**
4. 좌표 없는 다른 장소들도 30~150m 원형 분포

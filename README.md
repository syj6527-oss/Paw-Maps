# 🌍 World Tracker v0.8.8

> **장소 기반 커뮤니티 생성 RP 확장 — SillyTavern 전용**
>
> 한국 최초, 다른 확장엔 없는 독자 기능!

장소 이름만 넣으면 그 지역 주민들이 떠드는 **실시간 트위터 타임라인**이 자동 생성됩니다.
카자흐 사막부터 우리 동네 편의점까지 — AI가 **현지 고증**까지 해줘서 진짜 그 동네에 있는 느낌.

---

## ✨ 이 확장만의 킥 — "장소가 주인공"

다른 RP 도구들은 **캐릭터 중심**이에요. 이 확장은 **장소가 주인공**입니다.

- 🏛️ **장소 해시태그로 모인 익명 주민들의 피드**
- 👀 목격담 · 🍽️ 맛집 후기 · 🌆 동네 일상 · 🐱 길고양이 시점
- ✈️ RP 하면서 **간접 여행**하는 기분

### 🎯 실제 결과 예시

| 장소 | LLM이 뽑아낸 디테일 |
|---|---|
| 🇰🇿 카자흐 사막 | 라그만, 유목민, 사막여우 |
| 🇮🇹 나폴리 피자가게 | 마르게리따, 베수비오 산, 에스프레소 1분컷 |
| 🇰🇷 경복궁 | 근정전, 사직로, 꿀타래, 야간개장, 한복 |
| 🇰🇷 해운대시장 | **상국이네 떡볶이**(실존!), 부산 사투리 "마! 묵고 댕기나?", 밀면 |
| 🇰🇷 간절곶 | 소망우체통, 동해안 칼바람, **해빵**(Grounding 시) |

장소 이름 하나만 주면 현지 고증·사투리·명물·축제까지 알아서 뽑아냅니다.

## 🚀 주요 기능

### 🗺️ 장소 자동 감지
채팅하면 장소가 자동 등록됩니다. 한국어/영어 지원.
> "카페로 향했다" → 📍 카페 자동 등록

### 📍 Leaflet 지도 + 노드맵 약도
- 실제 세계 좌표 기반 지도 (Leaflet)
- 장소별 노드맵 약도 (내부 구조도)
- 걸어서 거리 계산 (1.4배 보정)

### 🟢 실시간 커뮤니티 피드 (핵심!)
장소마다 트위터 스타일 피드 생성:
- **7~9개 포스트** (설정으로 4~12개 조절 가능)
- **장소 특화 트윗 2~3개** — `#장소이름` 해시태그로 묶인 주민 반응
- **🌍 현지 색깔** — 나라/지역 특산물·문화·사투리
- **🔥 목격담** — "방금 {char}이 여기서 ~하던데..."
- **💬 답글** — 여러 주민이 엮이는 자연스러운 대화
- **🖼️ 이미지** — Pollinations.ai로 장소·음식·동물 사진 자동 첨부

### 🎨 여초 감성 말투 (30/40/30)
- **30% 담백** "오늘 노을 예뻤다."
- **40% 약한 감성** "이 동네 고양이 또 왔음 ㅋㅋ"
- **30% 폭발 감성** "미친 실화냐고 ㅋㅋㅋㅋㅋㅋ ㅠㅠㅠㅠ"
- 한국어 맞춤법 교정 (어떻게/어떡해 구분 등)

### 🔍 현지 정보 보강 (옵션)
설정 → 🔍 현지 정보 보강:
- **OFF** — 기본 (LLM 학습 데이터만)
- **🌐 무료 (Nominatim)** — 장소 500m 이내 실제 POI 수집
- **⭐ 고급 (Google Search Grounding)** — 실시간 웹 검색, 최신 정보까지 (유료)

### 📏 생성 분량 조절
비용/속도/풍성함 선택:
- 🌱 **가벼움** — 4~5개 (모바일 권장, 빠름)
- ⚖️ **기본** — 7~9개
- 🌿 **풍성함** — 10~12개

### 👥 NPC 터줏대감 시스템
장소별로 단골 NPC 자동 감지·등록. 호감도(1~5), 성격 태그, 관계 메모 관리.

### 📝 드래그→이벤트 저장
채팅에서 텍스트 드래그하면 AI가 한 줄 요약해서 **그 장소의 이벤트로 저장**.

### 🛠️ 커뮤니티 운영자 친화 기능
- **🐛 디버그 로그 뷰어** — 모바일에서도 에러 원인 확인 (F12 없이도 OK)
- **📋 복사 버튼** — 로그 붙여넣기 한 번으로 원인 진단
- 유저 버그 리포트가 편해짐

### 🎭 그룹챗 대응
여러 캐릭터 그룹챗에서도 한 트윗에 한 명만 자연스럽게 언급.
최근 발화자 자동 감지.

## 🔑 설정 방법

### 1. API 키 등록 (필수)
설정 → 🔑 LLM API 키 → 원하는 제공자 선택 후 키 입력:

- **Google AI Studio** (무료 1,500회/일) — 추천!
  - https://aistudio.google.com/apikey
- **Vertex AI** (Express mode, 서비스 계정 JSON 둘 다 지원)
- **OpenAI / OpenRouter / Claude** 지원

### 2. 모델 선택
- ⚡ **Gemini 2.5 Flash** (추천) — 빠르고 안정적
- 🧠 Gemini 2.5 Pro — 고품질 (느림, 토큰 많이 씀)
- 💨 Gemini 2.5 Flash Lite — 최저 비용

### 3. 생성 분량 / 현지 정보 보강 선택 (옵션)

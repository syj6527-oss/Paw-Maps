# 🐶 World Tracker

**AI 롤플레이를 위한 세계관 지도 확장**

RP 중 등장하는 장소를 자동으로 감지하고, 약도와 실제 지도에 정리해주는 확장 프로그램입니다.
채팅만 하면 알아서 세계관이 만들어져요.

---

## 이런 분들에게 추천해요

- RP 하다가 "아 여기 전에 왔던 곳인데..." 하고 까먹는 분
- 캐릭터가 어디있는지, 어디서 뭘 했는지 정리하고 싶은 분
- 구글맵 느낌으로 RP 세계관을 관리하고 싶은 분
- AI가 장소를 기억하고 분위기에 맞게 대화했으면 하는 분

---

## ✨ 주요 기능

### 🔍 자동 장소 감지
채팅만 하면 장소가 자동으로 등록됩니다. 한국어, 영어 모두 지원.
> *"카페로 향했다"* → 📍 카페 등록!
> *"headed to the market"* → 📍 Market 등록!

직접 등록할 필요 없어요. AI 응답에서 알아서 잡아줍니다.

### 🗺️ 노드 약도
장소들의 관계를 한눈에 볼 수 있는 약도가 자동으로 그려집니다.
- 거리 슬라이더로 장소 간 거리 설정
- 현재 위치, 자주 가는 곳, 새 장소를 색상으로 구분
- 롱프레스로 핀 위치 직접 이동
- 실제 지도 좌표를 약도에도 반영

### 🐾 Paw Maps (구글맵 스타일)
실제 지도 위에 RP 장소를 핀으로 표시합니다.
- 주소 검색 → 실제 좌표에 핀 배치
- 핀 터치 → 바텀시트로 장소 정보 확인
- 3단 바텀시트 (peek/half/full) + 스와이프 드래그

### 📊 개요 탭
장소를 탭하면 구글맵처럼 상세 정보가 나옵니다.
- **분위기 카드** — 장소 타입+시간대에 맞는 이모지가 자동 배치
- **방문 통계** — 방문 횟수, 첫/최근 방문 시간
- **주변 장소** — 도보 15분 이내 장소 표시
- **리뷰 미리보기** — AI가 생성한 캐릭터 리뷰
- **최근 방문 기록 / 기억 링크**

### ⭐ AI 랜덤 리뷰
장소마다 AI가 RP 캐릭터 시점의 구글맵 스타일 리뷰를 생성합니다.
- 방문 횟수에 따라 1~10개 리뷰 (많아질수록 확률 감소)
- 캐릭터 이름, 성격, 별점이 랜덤 생성
- RP 이벤트를 반영한 리뷰 내용

### 📝 이벤트 자동 기록
키스, 싸움, 약속, 비밀 발견... RP에서 일어나는 사건을 자동으로 감지하고 기록합니다.
- 80+개 키워드 (한/영) 자동 감지
- AI(LLM)가 분위기에 맞는 제목+요약 생성
- 텍스트 드래그 → 📝 버튼으로 수동 저장도 가능

### 🕐 타임라인
날짜별 이동 기록을 한눈에 볼 수 있습니다.
- 오늘은 펼쳐진 상태, 과거는 아코디언으로 접힘
- 이동 경로 + 이벤트 태그 표시
- 날짜별 통계 ("3곳 방문 · 이벤트 2건")

### 🔖 내 페이지
총 장소 수, 총 방문 횟수, 최다 방문 장소를 한눈에 확인.
최근 방문한 장소 목록도 바로 접근 가능합니다.

### 🤖 AI 프롬프트 주입
현재 위치, 방문 횟수, 주변 장소 정보를 AI에게 자동으로 전달합니다.
덕분에 AI가 장소를 기억하고, 현실적인 이동과 분위기를 유지해요.

### 🏰 판타지 모드
판타지 세계관용 아이콘 세트와 분위기로 전환됩니다.
성, 산, 숲, 동굴, 항구 등 14종 아이콘 지원.

## 📦 설치

1. SillyTavern `data/default-user/extensions/third-party/` 폴더에 복사
2. SillyTavern 재시작
3. 확장 프로그램 목록에서 **🐶 World Tracker** 활성화
4. 채팅 화면에서 🐶 버튼으로 패널 열기

## 🎮 사용법

**기본: 그냥 RP 하세요.** 장소 감지, 이벤트 기록, 프롬프트 주입 전부 자동입니다.

필요할 때만:
- 패널에서 장소 터치 → 메모, 별칭, 거리 설정
- Paw Maps → 주소 검색으로 실제 좌표 연결
- 이벤트 탭에서 기록 확인/수정

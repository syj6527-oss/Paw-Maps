# 🐶 World Tracker v0.3.0-beta

**SillyTavern RP 월드 트래커 확장 프로그램**

AI 응답에서 장소를 자동 감지하고, 약도 + 🐾 Paw Map(실제 지도)에 표시하는 RP 세계관 관리 도구입니다.

---

## ✨ 주요 기능

### 🔍 자동 장소 감지
- AI 응답과 유저 입력에서 **장소 자동 인식** (한국어/영어)
- 이동 동사 기반 감지: "카페로 향했다", "headed to the market"
- USER/AI 감도 분리 — AI는 엄격 모드로 오탐 방지
- 기존 채팅에 설치 시 → **최근 메시지 스캔 + 승인 UI**

### 🗺️ 약도 (플랫 시티맵)
- Hub 기반 거점 이동식 지도 — 도보 15분 이내만 표시
- 비대칭 블록 그리드 + 대로/골목 차등 도로
- 강 + 호수 + 공원 자동 생성
- 장소별 컬러 핀 + 이모지 아이콘 (15종)
- 방문 횟수 뱃지 + 현재 위치 🐾 애니메이션
- 🏰 판타지 모드 (양피지 배경 + 나침반)

### 🐾 Paw Map (구글맵 스타일 실제지도)
- OpenStreetMap + CartoDB Voyager 타일
- **구글맵 물방울 핀** — SVG teardrop + 장소 이모지, 선택=빨강, 현재=초록
- **풀스크린 모드** — 반투명 플로팅 검색/탭 (backdrop-filter:blur)
- **장소/주소 듀얼 검색** — Nominatim 지오코딩
- **🐾 Paw Map 태그** — 좌하단 워터마크

### 📋 바텀시트 (구글맵 3단계)
- **Peek** (185px) → **Half** (55%) → **Full** (전체) 3단계
- 핸들바 **클릭** + **터치 드래그** 둘 다 지원
- 손 떼면 가장 가까운 단계로 **자석 스냅**
- Full에서 핸들 클릭 → 완전 닫기
- 외부 터치 → 닫기

**개요 탭:**
- 📍 실제 주소
- 📊 방문 횟수 (첫/최근)
- 🤖 특이사항 (AI에게만 전달)
- 📌 주변 장소 (도보 15분 이내)

**이벤트 탭:**
- 🌡️ 분위기 지수 막대 그래프
- 이벤트 카드 (아코디언 펼침/접힘)

**리뷰 탭:**
- 🤖 AI 리뷰 요약 (시적 한 문장)
- ⭐ 별점 5단계 가로 막대 그래프
- 감정 키워드 칩 (💕로맨틱/⚡긴장/☕커피)
- 캐릭터/유저/NPC/동물 랜덤 리뷰 1~3개
- IndexedDB 캐시 (재방문 시 즉시 표시)

### 🔖 하단 탭
- 🐾 **탐색** — 지도 보기 (핀 터치 → 바텀시트)
- 🔖 **내 페이지** — 통계칩 + 최근방문 + 내 목록(💜⭐🚩🧳)
- 🕐 **타임라인** — (개발 예정)

### 🤖 AI 프롬프트 주입
- 현재 위치, 상태, 방문 횟수 자동 주입
- 특이사항(aiNotes) + 이벤트 + 근처 장소 포함

```
[🐶 World Tracker]
📍 Scene: 삐삐카페
📊 Visit #3
💭 김홍진의 단골카페
📋 Notes: 0900 붐빔, 바리스타 민수, 2층 창가석 단골
📝 Memories here:
- 💕 첫 데이트의 기적
- 📅 반려동물 키우기 의견충돌
🚶 Last: 집 → 삐삐카페 (도보 10분)
📌 Nearby:
- 집 (도보 10분)
- 편의점 (바로 옆)
[/World Tracker]
```

### 📏 거리 시스템

- RP 친화적: 바로 옆 / 도보권 / 차량 필요 / 다른 지역

# 🐶 World Tracker v0.3.0-beta

**SillyTavern RP 월드 트래커 확장 프로그램**

AI 응답에서 장소를 자동 감지하고, 플랫 시티맵 약도 + 실제 지도에 표시하는 RP 세계관 관리 도구입니다.

---

## ✨ 주요 기능

### 🔍 자동 장소 감지
- AI 응답과 유저 입력에서 **장소 자동 인식** (한국어/영어)
- 이동 동사 기반 감지: "카페로 향했다", "headed to the market"
- USER/AI 감도 분리 — AI는 엄격 모드로 오탐 방지
- 기존 채팅에 설치 시 → **최근 메시지 스캔 + 승인 UI**

### 🗺️ 플랫 시티맵 약도 (v0.3.0 신규)
- **Hub 기반 거점 이동식 지도** — 도보 15분 이내 = 같은 동네, 그 이상은 별도 Hub
- 비대칭 rect 그리드 + 블록 병합 → 실제 도시 구획 느낌
- **대로/골목 차등 도로** — 3칸마다 넓은 대로(26px), 나머지 좁은 골목(8px)
- 주황+노랑 **간선도로 십자가** 교차 배치
- **강** — 블록 밑 레이어, 일직선 관통
- 호수(둥근 네모) + 공원(나무/연못) 자동 생성
- **clipPath 액자 프레임** — 가장자리 깔끔하게 클리핑
- 카메라(ViewBox) 분리 — 핀 클릭 시 카메라만 이동, 배경 재생성 X
- 장소별 컬러 핀 + 이모지 아이콘 (🐱카페/📚서점/🏠집/🌳공원 등 15종)
- 방문 횟수 뱃지 + 현재 위치 🐾 애니메이션
- 거리 점선 + 흰색 pill 뱃지
- 롱프레스로 핀 드래그 이동

### 🏰 판타지 모드
- 양피지 배경 + 세리프 폰트 + 나침반 로즈
- 이모지 14종 아이콘 (🏰성/⛰️산/🌲숲/⚓항구 등)
- 🐶→🐺 마스코트 전환, 🦴→🍖 트릿 전환

### 🌍 실제 지도 (Leaflet)
- OpenStreetMap 기반 실제 지도
- **주소 검색** → Nominatim 지오코딩 → 핀 배치
- 좌표 기반 방향 보존 (실제 북/남/동/서 → 약도에도 반영)
- 커스텀 컬러 마커 + 리치 팝업

### 🤖 AI 프롬프트 주입
- 현재 위치, 상태, 방문 횟수 자동 주입
- **근처 장소 + RP 거리 정보** 포함
- AI가 이동 현실성을 유지하도록 가이드

```
[🐶 World Tracker]
📍 Scene: 고양이 카페
📊 Visit #12
💭 따뜻한 라떼와 고양이들
🚶 Last: 별빛서점 → 고양이 카페 (도보 8분)
📌 Nearby:
- 별빛서점 (0.8km)
- 중앙공원 (도보 8분)
- 루미네 집 (도보 12분)
[/World Tracker]
```

### 📏 거리 시스템
- **자유 텍스트** + **레벨 슬라이더 (1~10)** 병행
- RP 친화적 라벨: 바로 옆 / 매우 가까움 / 도보권 / 차량 필요 / 다른 지역
- "골목 두 개 건너", "지하 터널 5분" 같은 세계관 표현 가능

### 🐾 추가 기능
- 커스텀 알림 시스템 (토스트)
- 한/영 이중 등록 방지 (집↔Home 사전 28그룹)
- 별칭 관리 (쉼표 구분)

- 이동 히스토리 (삭제 가능)
- 기억 모드: 💎완벽(영구) / 🌿자연(시간 경과 흐림)
- 이벤트 자동 추출 + 드래그 저장
- 채팅별 백업/복원/삭제

## 🎮 사용법

### 처음 사용
1. RP 채팅 시작 → 자동으로 장소 감지
2. 감지된 장소 **승인 UI**에서 확인/수정/등록
3. 패널에서 장소 터치 → 메모/상태/별칭/거리 설정

### 약도 만들기
1. 장소 2개 이상 등록
2. 장소 터치 → 거리 슬라이더로 다른 장소와의 거리 설정
3. **🗺️ 약도** 버튼 클릭 → 플랫 시티맵 자동 생성!

### 실제 지도 핀 찍기
1. 장소 터치 → 팝오버 하단 **🔍 주소 검색**
2. "서울 서면" 또는 "Brooklyn" 입력 → 검색
3. 결과 터치 → **자동으로 Leaflet 전환 + 핀 배치**

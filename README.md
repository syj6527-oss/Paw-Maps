# 🐶 World Tracker v0.5.0

**RP하면 알아서 채워지는 자동 로어북**

채팅만 하면 장소가 자동으로 지도에 정리되고, AI가 그 정보를 기억해서 대화에 활용합니다.
설정할 것도 외울 것도 없어요. 그냥 RP 하세요!

---

## 이런 분들에게 추천해요

- RP 하다가 "여기 전에 왔던 곳인데..." 하고 까먹는 분
- 캐릭터가 어디있는지, 어디서 뭘 했는지 정리하고 싶은 분
- 구글맵 느낌으로 RP 세계관을 꾸미고 싶은 분
- AI가 장소를 기억하고 분위기에 맞게 대화했으면 하는 분
- NPC들의 호감도와 관계를 추적하고 싶은 분

---

## ✨ 주요 기능

### 🔍 장소 자동 감지
채팅만 하면 장소가 자동 등록됩니다. 한국어, 영어 모두 지원.
> "카페로 향했다" → 📍 카페 등록!
> "headed to the market" → 📍 Market 등록!

### 👥 터줏대감 시스템 (v0.5.0 NEW!)
장소마다 등장하는 NPC와 동물을 자동 감지하고 프로필을 관리합니다.
- **자동 감지**: RP 중 등장하는 NPC/동물 자동 등록
- **호감도 시스템**: ❤️ 1~5단계 — 채팅 분위기에 따라 자동 변동 (다정하면 ↑, 갈등이면 ↓)
- **프로필 카드**: 터치하면 상세 프로필 (아바타/성격/관계/최근 이벤트)
- **AI 프로필 생성**: 🔄 버튼으로 LLM이 채팅 맥락 기반 프로필 자동 생성
- **수동 편집**: 이름, 역할, 아바타, 성격 태그, 관계 메모 직접 수정
- **리뷰 연동**: 리뷰에 등장한 NPC도 자동 등록
- **프롬프트 주입**: NPC 정보(호감도/관계)가 AI 컨텍스트에 포함되어 캐릭터 일관성 유지

### 🗓️ 예정 일정 시스템 (v0.5.0)
RP 대화에서 약속이나 일정을 자동으로 추출합니다.
- "내일 마트 가자" → 🗓️ 일정 자동 등록
- LLM이 미래 약속 정밀 추출 (future_plan)
- RP 날짜 기반 실제 날짜 계산
- 회색 카드로 시각적 구분

### 📷 사진 갤러리 (v0.5.0)
장소별 사진을 저장하고 구글맵 스타일로 보여줍니다.
- 장소당 최대 5장, JPEG 70% 압축
- 그리드 레이아웃 + "+N" 오버레이
- 풀스크린 뷰어, 개별 삭제

### 🗺️ 약도 + 실제 지도 (Paw Maps)
- **약도**: 장소 관계를 한눈에 보는 노드 맵
- **Paw Maps**: 실제 지도 위에 핀으로 표시 (구글맵 스타일)
- 주소 검색 → 실제 좌표에 배치
- 핀 터치 → 장소 상세 정보 (바텀시트)

### 📍 자동 주소 + 거리 계산
첫 장소의 주소만 검색하면, 이후 장소들은 알아서 근처에 배치됩니다.
- 새 장소 → 기존 장소 근처에 자동 좌표 배치
- 좌표 기반 실제 거리 자동 계산 (도보 몇 분)
- 주소도 자동 부여!

### 🏠 내부 장소 (장소 계층)
집 안의 거실, 부엌, 방 같은 세부 장소를 자동으로 구분합니다.
- AI가 "거실로 갔다" → 집 > 거실
- 3중 분리: 대시 / 공백 키워드 / 층수
- 서브장소 이름 변경 가능 (v0.5.0 NEW!)
- 각 방마다 독립적인 이벤트 기록

### 🤖 AI 프롬프트 자동 주입
현재 위치, 주소, 방문 기록, 이벤트, NPC 정보를 AI에게 자동으로 알려줍니다.

포함 정보:
- 현재 장소 이름 + 주소 + 타입
- 방문 횟수 + 첫 방문 날짜
- 최근 이벤트 (날짜 포함)
- 주변 장소 + 거리 + 주소
- 내부 장소 (거실, 부엌 등)
- 👥 터줏대감 프로필 (호감도/관계) (v0.5.0 NEW!)
- 세계관 지역 자동 추론

### 🔀 AI 중복 등록 방지
유저가 "마트"라고 하고 AI가 "창신동 슈퍼마켓"이라고 하면,
같은 곳인지 자동으로 물어봅니다.
- 별칭 키워드 매칭 강화 (v0.5.0 NEW!)
- "SAS 북부 무기고" → 별칭 "무기고" 히트

### 📊 바텀시트 (구글맵 스타일)
장소를 탭하면 상세 정보가 나옵니다.
- **개요**: 분위기 카드, 방문 통계, 주변 장소, 리뷰/NPC 미리보기
- **이벤트**: 자동 감지된 사건들 + 예정 일정
- **리뷰**: AI가 만든 캐릭터 시점 리뷰 (구글맵 스타일)
- **터줏대감**: NPC/동물 목록 + 호감도 + 프로필 카드 (v0.5.0 NEW!)
- **내부**: 건물 안 세부 장소 목록

### ⭐ AI 캐릭터 리뷰 (v0.5.0 강화!)
장소마다 AI가 RP 캐릭터 시점의 리뷰를 만들어줍니다.
- 방문 횟수에 따라 3~7개 (최소 3개 보장)
- 터줏대감이 우선 리뷰어로 등장
- JSON 파싱 강화 + 자동 재시도로 안정성 향상
- 리뷰어 → 터줏대감 자동 등록

### 📝 이벤트 자동 기록 (v0.5.0 강화!)
RP에서 일어나는 사건을 자동으로 감지하고 기록합니다.
- AI가 분위기에 맞는 요약 생성
- 화자 식별 강화 — 대화 구조 분석으로 화자 뒤바뀜 감소
- NPC 호감도 자동 변동 (npc_interactions)

### 🕐 타임라인
날짜별 이동 기록을 한눈에 볼 수 있습니다.
- RP 날짜 동기화 (rpDate 우선 그룹핑)
- Celia/P&C 날짜 형식 호환

### 🔖 내 페이지 + 태그
장소를 카테고리별로 정리할 수 있습니다.
- 💜 즐겨찾기 / ⭐ 별표 / 🚩 가고싶은곳 / 🧳 여행계획

### 🌍 세계관 이어가기
새 채팅을 만들어도 같은 캐릭터의 세계관을 유지합니다.

### 🏰 판타지 모드
판타지 세계관용 아이콘과 분위기로 전환됩니다.

## 🎮 사용법

- 장소 터치 → 메모, 별칭, 거리 설정
